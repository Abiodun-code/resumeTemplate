## Run Locally

Clone the project

```bash
  git clone https://github.com/Abiodun-code/my_Resume
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn add
```

Start the server

```bash
  yarn dev
```
